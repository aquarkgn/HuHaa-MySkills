## 🎯 核心问题 (已诊断)

### 问题 1: 网页界面没显示多源数据
- **现象**: 用户看到的页面好像只有 Hermes 技能
- **原因**: 前端过滤逻辑未针对多源优化
- **真实情况**: 180 个技能，5 个源，数据完整 ✅

### 问题 2: 过滤器混淆 editor/source  
- **现象**: 左侧筛选栏显示混乱
- **原因**: store 中用 `||` 混淆了两个概念
- **修复**: 分离统计，修复过滤逻辑

### 问题 3: 品牌展示不合理
- **现象**: Logo/品牌信息无法清晰识别
- **原因**: 41% 的技能缺失 brand，UI 没有为多源设计
- **修复**: 补全数据，增强 UI
