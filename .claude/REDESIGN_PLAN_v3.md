## 📐 具体实现

### 1. **顶部搜索和过滤区**（保留和改进）
```
[🔍 搜索框 ...]  [清空]  [扫描]

✓ Hermes  ✓ Claude Code  [+更多]  [清除过滤]
```

### 2. **技能列表区**（紧凑设计）
- Grid 布局：4-6 列（响应式）
- 卡片尺寸：小（120x100px ~ 160x130px）
- 卡片内容：
  - 图标 + 名称（2 行）
  - 来源标签
- hover/selected 效果：突出显示
- **点击卡片 → 打开全屏弹窗**

### 3. **技能详情弹窗**（全屏）
```
┌─────────────────────────────────────┐
│ 技能名称                    [✕ 关闭]│
├─────────────────────────────────────┤
│                                     │
│  描述                               │
│                                     │
│  📁 路径: /path/to/skill  [复制]    │
│  🏷️ 标签: skill | hermes | ...     │
│  📊 信息: 来源、编辑器、大小等      │
│                                     │
│  💡 使用方法                        │
│  skill_view(name='xxx')             │
│                                     │
│  📄 完整内容                        │
│  ────────────────────────────       │
│  (Markdown 渲染)                    │
│  可滚动                             │
│                                     │
└─────────────────────────────────────┘
```

### 4. **交互流程**
1. 用户在列表中选择技能
2. **全屏黑色背景遮罩**出现
3. **白色弹窗**在中心展示
4. 用户可以：
   - 滚动查看完整内容
   - 复制路径
   - 点击关闭按钮或背景关闭
   - 按 ESC 关闭

## 🔧 实现细节

### App.vue 结构
```vue
<template>
  <div class="app-container">
    <!-- 顶部搜索和过滤 -->
    <header class="app-header">...</header>
    
    <!-- 技能列表（紧凑网格） -->
    <main class="app-main">
      <div class="skills-grid">
        <div 
          v-for="skill in filtered"
          class="skill-card-compact"
          @click="selectSkill(skill)"
        >
          <!-- 紧凑内容 -->
        </div>
      </div>
    </main>

    <!-- 全屏弹窗 -->
    <div v-if="selected" class="modal-overlay" @click.self="closeDetail">
      <div class="modal-dialog">
        <!-- 完整的技能详情 -->
      </div>
    </div>
  </div>
</template>
```

### 样式关键点
- 列表卡片：100-150px 小卡片
- 弹窗：95vw × 90vh（最大宽度 900px）
- 背景遮罩：`background: rgba(0, 0, 0, 0.6)`
- 动画：淡入 + 弹窗滑入
