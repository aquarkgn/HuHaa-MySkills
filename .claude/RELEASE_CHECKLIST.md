Release v0.3.3 — 页面布局改版

五阶段改版完成，生产就绪。

核心改进:
  • Sidebar 扩宽 20% (200px → 240px)
  • List 面积增加 180% (600px → 1680px)
  • Detail Panel 改为浮层模态框
  • Filter Chips 新增显示与交互
  • 5 个完整的响应式断点 (375px-1920px+)
  • 卡片网格化 (1-5 列自动适应)

技术细节:
  • Vue 3 Teleport 组件
  • CSS Grid auto-fill 自适应布局
  • 纯 CSS 动画 (GPU 加速)
  • 现代浏览器完整支持
