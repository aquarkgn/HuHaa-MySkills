# SkillHelper 2026 Q3 抖音推广计划

## 目标

用真实产品演示建立 **“SkillHelper = 本地 AI 工作流能力中枢”** 的认知，并把合格流量导向 GitHub、npm 安装和后续教程。

本期只做抖音。先跑通“钩子 -> 录屏证据 -> 评论转化 -> 复盘”的闭环，再考虑扩展平台。

## 核心叙事

**痛点**：AI 技能、插件和 MCP 配置分散在多个工具中，难以发现、检索和维护。

**方案**：SkillHelper 扫描本机能力来源，按层级和来源统一展示，并提供搜索、筛选、原文/中文对照与图标识别。

**证据**：真实命令启动、真实面板、真实筛选交互、真实 API/图标表现；每项由镜头清单中的 `evidence` 支撑。

**CTA**：评论关键词、主页置顶仓库、关注下一集。每条视频只保留一个主动作。

## 当前口径

- 产品名：`SkillHelper`。
- 中文说明：呼哈哈-技能助手。
- npm / CLI：当前仍使用 `skillshelper`。
- 录制当天必须重新复核 npm latest、GitHub 页面和实际 UI。

```bash
npm install -g skillshelper@latest
skillshelper --version
skillshelper init
skillshelper scan --json
```
